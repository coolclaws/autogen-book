# 附录 A 推荐阅读路径

> 本书共 20 章加 3 个附录，内容涵盖 AG2 的架构设计、核心源码和实践模式。不同背景的读者可以选择不同的阅读路径，以最高效的方式掌握所需知识。

## A.1 入门路径：从零开始构建多智能体应用

适合刚接触 AG2 或多智能体框架的开发者。目标是快速理解核心概念并能编写基础的多智能体工作流。

**推荐顺序：**

```
第 1 章（项目全景）
  → 第 3 章（ConversableAgent 核心）
    → 第 4 章（消息机制）
      → 第 5 章（回复函数链）
        → 第 8 章（GroupChat）
```

| 章节 | 你将学到 | 预计时间 |
|------|----------|----------|
| 第 1 章 | AG2 的定位、核心概念和整体架构 | 30 分钟 |
| 第 3 章 | ConversableAgent 的构造参数和生命周期 | 45 分钟 |
| 第 4 章 | Agent 之间如何发送和接收消息 | 45 分钟 |
| 第 5 章 | 回复函数链的注册、排序和执行机制 | 60 分钟 |
| 第 8 章 | 多 Agent 群聊的选人策略和消息广播 | 60 分钟 |

完成入门路径后，建议阅读第 18 章（对话历史管理）和第 19 章（记忆系统），补充生产环境必需的知识。

| 章节 | 你将学到 | 前置要求 |

## A.5 阅读建议

1. **不必按顺序通读**——本书每章相对独立，在前置知识满足的前提下可以跳读。
2. **边读边实验**——建议克隆 AG2 仓库，对照源码阅读。关键代码片段均标注了文件路径和行号。
3. **关注 `_` 前缀方法**——AG2 的许多核心逻辑在以下划线开头的"私有"方法中，阅读时不要跳过。
4. **善用附录 B**——在阅读任何章节时，附录 B 的类型速查表都可以作为随手翻阅的参考。
